# Localee — нативное iOS-приложение (SwiftUI)

Полностью нативное приложение на Swift/SwiftUI. Ходит на тот же бэкенд, что и сайт
(`https://api.localee.ru`). Никаких node_modules, Metro и дев-серверов.

## Как открыть и запустить

```bash
open ios-swift/Localee.xcodeproj
```

В Xcode: выбрать сверху **iPhone 17 Pro** (или любой iPhone) → нажать **▶**.

## Экраны
- **Auth** — вход/регистрация (реальные аккаунты сервера)
- **Карта** — MapKit + места по категориям, фильтры, шторка снизу как в Яндексе,
  18+, пользовательские метки (скопления/сходки/дрифт)
- **Лента** — посты, лайки, публикация
- **Чаты** — список диалогов + переписка
- **Профиль** — данные + выход

Светлая/тёмная тема — по системной.

## Разработка
Проект генерируется из `project.yml` через [XcodeGen](https://github.com/yonaskolb/XcodeGen).
При добавлении **нового** `.swift`-файла: `cd ios-swift && xcodegen generate`.
Сам `Localee.xcodeproj` закоммичен — чтобы открывать без установки XcodeGen.

## Симулятор: если «hostname could not be found»
DNS симулятора иногда не резолвит домены. Разово на Mac:
```bash
echo "80.78.244.37 api.localee.ru" | sudo tee -a /etc/hosts
```
На реальном iPhone не требуется.
